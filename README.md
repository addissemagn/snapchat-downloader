# Setting up
You can use SnapSaver on the website or on your machine. To setup locally: 

*****Clone the repo**
```
git clone git@github.com:addissemagn/snapsaver.git
cd snapsaver
```

**Initialize a virtualenv**
```
pip install virtualenv
virtualenv env
source env/bin/activate
```
